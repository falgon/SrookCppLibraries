## Recommendation
* Ubuntu 16.04 or later
* Mac OS Sierra or later

### Compiler
* Clang 5.0.0 or later
* GCC 6.3.0 or later
* Apple LLVM Clang 800.0.42.1 or later

## Recommendation
Boost C++ Libraries version 1.5.8 or later
(This library often uses boost-c++, but works without it.)

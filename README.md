# Check on
## OS
* Ubuntu 16.04
* Mac OS Sierra

## Compiler
* Clang 5.0.0 Head
* GCC 6.3.0
* Apple LLVM Clang 800.0.42.1

# Recommendation
Boost C++ Libraries version after 1.5.8
(This library often uses boost-c++, but works without it.)
